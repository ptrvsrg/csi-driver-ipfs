# Maintainers

This file lists the maintainers of the csi-driver-ipfs project and how to contact them.

## Current maintainers

| Name    | GitHub                                 | Responsibilities                    |
|---------|----------------------------------------|-------------------------------------|
| ptrvsrg | [@ptrvsrg](https://github.com/ptrvsrg) | Project lead, code review, releases |

## How to become a maintainer

- Contribute consistently (code, docs, or reviews).
- Be active in issues and pull requests.
- Existing maintainers may propose adding new maintainers; consensus is required.

## Contact

- **Issues:** [GitHub Issues](https://github.com/ptrvsrg/csi-driver-ipfs/issues)
- **Discussions:** [GitHub Discussions](https://github.com/ptrvsrg/csi-driver-ipfs/discussions) (if enabled)

For security-sensitive matters, contact the maintainers privately (e.g. via GitHub security advisories or the email
visible in commit history).
